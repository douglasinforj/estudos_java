# Sistema Escolar - Java POO (Projeto Estudos)

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de Programação Orientada a Objetos (POO) em Java, além de aplicar boas práticas de organização de código utilizadas no mercado.

A aplicação simula um sistema escolar simples, com cadastro de alunos e professores, cálculo de média e persistência de dados em arquivo .csv.

## Tecnologias utilizadas
 - Java (JDK 8+)
 - VSCode
 - Programação Orientada a Objetos (POO)
 - Manipulação de arquivos (NIO)
